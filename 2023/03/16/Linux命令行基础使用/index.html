关于Linux命令行的基本使用。

Vim

Vim有四种模式

  • 普通模式，其他模式按esc
  • 插入模式，普通模式按i, I, a, A等键
  • 可视模式，普通模式按v, V, ctrl+v，用于选中文本。
  • 命令行模式，普通模式按冒号，以下是退出编辑器和保存文件的常用命令。
    • q：如果未修改缓冲区数据，退出。
    • q!：取消所有对缓冲区数据的修改并退出。
    • w filename：将文件保存到另一个文件中。
    • wq：将缓冲区数据保存到文件中并退出。

编辑数据

普通模式可以使用以下的编辑命令（普通模式下直接输入即可）

  • x 删除当前光标所在位置的字符
  • dd 删除当前光标所在行
  • dw 删除当前光标所在位置的单词
  • d删除当前光标所在位置至行尾的内容
  • J 删除当前光标所在行行尾的换行符（拼接行）
  • u 撤销前一编辑命令
  • a 在当前光标后追加数据
  • A 在当前光标所在行行尾追加数据
  • r char 用char替换当前光标所在位置的单个字符
  • R text 用text覆盖当前光标所在位置的数据，直到按下ESC键

复制和粘贴

复制命令是y（代表yank），可以在y后面使用和d命令相同
的第二字符（yw表示复制一个单词，y$表示复制到行尾）。在复制文本后，把光标移动到你想放置文本的地方，输入p命令。复制的文本就会出现在该位置。

可以用可视模式来选取要复制的区域，选取后直接按y即可复制指定。

查找和替换

查找：按下划线后输入查找字符串，使用n键，表示查找下一个（next）。查找成功会高亮文本，使用:nohl（no hightlight）命令可以取消高亮。

替换：:s/old/new/

  • :s/old/new/g：一行命令替换所有old。
  • :n,ms/old/new/g：替换行号n和m之间所有old。
  • :%s/old/new/g：替换整个文件中的所有old。
  • :%s/old/new/gc：替换整个文件中的所有old，但在每次出现时提示。

基本配置

  1. 配置用户名和邮箱

  2. 配置SSH

    • 生成 SSH Key ，输入命令：
      ssh-keygen -t rsa -C "youremail@example.com"
      之后点回车

    • 目录 .ssh 下，会生成两个文件 id_rsa(密匙) 和 id_rsa.pub(公匙)

    • 在代码托管网站（github）- Add SSH Key ：粘贴 id_rsa.pub(公匙) 内容

  3. 使用配置文件，位于.git/config目录下
