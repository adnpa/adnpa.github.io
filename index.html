数据类型

属性和度量

数据对象，也可称为记录、点、向量、模式、事件、案例、样本、观测或实体。用一组刻画对象基本特性的属性描述。属性也称为变量、特性字典、特征或维。

属性（attribute）是对象的性质或特性，例如物体的长度、质量等。需要注意的是：属性并非是符号或数字。

测量标度（measurement scale）是将数值或符号值与对象的属性相关联的规则（函数）。
为了精确地分析属性，需要人为地赋予属性对应的符号或数值，例如米的长度被定义为“光在真空中于1/299792458秒内行进的距离”。

属性的类型

数值（符号也可用数字代表）的性质被用于描述属性

  • 相异性 $=$ 和 $\neq$
  • 序 $<$、$>$、$\le$、$\ge$
  • 加法 $+$、$-$
  • 乘法 $\times$ 、${\div}$

标称（nominal），能够提供区分对象的信息，例：雇员ID、性别等

序数（ordinal），用于表示定性的大小关系（序），例：矿石硬度（好，较好，最好）、成绩等
用 [1, 2, 3] 和 [4, 5, 6] 都可以表示好，较好，最好

区间（interval），值之间的差 （$+$、$-$）有意义，

比率（ratio），差和比率有意义（ $\times$ 、${\div}$）

标称和序数被称为分类或定性属性，而区间和比率被称为数值或定量属性

按值的个数描述

  • 离散的（discrete）
  • 连续的（continuous）

非对称的属性（asymmetric attribute）

数据集的类型

  • 维度（dimensionality）
  • 稀疏性（sparsity）
  • 分辨率（resolution）

数据挖掘概念

数据挖掘的出现是为了解决

  • 可伸缩，数据挖掘算法需要具备可伸缩性，即能处理指数级问题
  • 高维度性，处理具有成百上千的数据集
  • 异种数据和复杂数据
  • 数据的所有权与分布，需要分析的数据地理上分布在属于多个机构的资源 -> 分布式数据挖掘技术
  • 非传统的分析，传统分析基于假设-检验：提出假设，设计实验收集数据，针对假设分析数据。而当前数据分析任务可能设计数千种假设，因此促使人们发明一种自动假设和评估的算法
